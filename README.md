# FatScreen

### Inputs
- analog audio (mic/cassette player)
- RCA (consoles/VHS) * 8
- hdmi 
- RF cable (TV antena)
- pseye (camera + mic)
- ethernet
- usb keyboard/remote
- pi camera

### Outputs
- RCA (old TVs)
- hdmi (new TVs)
- audio jack (external speaker)
- ethernet (go online with playstation 2)
- dial up modem (go online with dreamcast)
- usb port for charging remote/bluetooth speaker
